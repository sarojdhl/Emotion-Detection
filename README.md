# Emotion-Detection

# To train and save model

1.  Go to Emotion_Detection folder
2.  Run python -m src.main

# To run api

1.  Go to folder 'api'
2.  Run python app.py
3.  Goto http://127.0.0.1:5000/

# To run dockerfile

1.  docker build -t my_img
2.  docker run -it --name my_container -p 5000:5000 my_img
